## âœ¨ The App in 60 Seconds

**What it does:**

- Teaches 8 psychology concepts through interactive 3D experiences
- Two modes: Guided (structured learning path) or Free (explore at your own pace)
- Beautiful, artistic aesthetic with premium design
- Fully accessible (keyboard, screen readers, WCAG AA)
- Works on mobile, tablet, desktop
- Graceful fallback for older browsers

**How it works:**

1. User lands on beautiful hero homepage
2. User chooses: Explore (guided) or Wander (free)
3. User browses psychology chapters
4. User clicks chapter to see full content
5. 3D scene renders for chapter topic
6. User reads description and long form content
7. User reflects on prompts (saved locally)
8. User reads peer-reviewed citations with DOI links
9. User navigates to next chapter

**Who it's for:**

- Psychology students
- Self-improvement seekers
- Educators
- Anyone curious about human behavior
